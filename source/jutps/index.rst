JUTPS
=====

Overview
--------

JUTPS is a character controller system designed for third-person and first-person games.
It supports both PC and Android platforms and is built to be flexible, extensible, and easy to integrate
into different types of projects.

The system focuses on providing a solid and consistent movement features.
